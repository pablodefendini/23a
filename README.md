# 23a — Marcha por la Justicia Reproductiva

Landing page for the march for reproductive justice on April 23, 2026 in Río Piedras, Puerto Rico.

Static site — one HTML file, one CSS file, custom fonts, and an SVG lockup. No JavaScript, no build step.

## Local development

Open `index.html` in a browser. That's it.

For live reload during development, use any static file server:

```bash
python3 -m http.server 8000
# or
npx serve .
```
